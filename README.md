# ⚡ FPS Monitor — Мониторинг Частоты Кадров Экранного Overlay

[![Android](https://img.shields.io/badge/Platform-Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)](https://android.com)
[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.x-7F52FF?style=for-the-badge&logo=kotlin&logoColor=white)](https://kotlinlang.org)
[![Compose](https://img.shields.io/badge/Jetpack_Compose-M3-4285F4?style=for-the-badge&logo=jetpackcompose&logoColor=white)](https://developer.android.com/jetpack/compose)
[![Build APK](https://img.shields.io/badge/GitHub_Actions-CI-2088FF?style=for-the-badge&logo=githubactions&logoColor=white)](https://github.com)

**FPS Monitor** — это высокопроизводительное системное приложение для операционной системы Android, разработанное на языках **Kotlin** и **Jetpack Compose (Material 3)**. Приложение позволяет в реальном времени отслеживать частоту кадров (FPS) внутри интерфейса и поверх абсолютно любых посторонних приложений или тяжелых Android-игр.

Приложение включает два альтернативных метода отрисовки, интерактивный бенчмарк-симулятор физических частиц для искусственного нагрева процессора и подробные графики телеметрии.

---

## ✨ Основные возможности программы

*   **⚡ Двойной алгоритм Overlay-виджета**:
    *   **Основные Спец. Возможности (`Accessibility Service`)**: Максимально точный хук отрисовки дисплея. Позволяет выводить компактный, стильный плавающий индикатор FPS в углу экрана поверх любой игры. Плашку можно свободно зажимать и перетаскивать пальцем по экрану.
    *   **Классическое наложение (`System Overlay`)**: Традиционный резервный вариант "Поверх других окон" на случай, если вы не хотите давать доступ спец. возможностям.
*   **📈 Графический дашборд реального времени**:
    *   Точный спидометр/полукруг с анимацией задержек до 120+ FPS.
    *   Автоматический расчёт среднего времени кадра в миллисекундах (`ms/кадр`).
    *   Плавный линейный график истории изменения частоты с сеткой порогов (30/60/90/120 FPS).
*   **📊 Расширенная игровая статистика**:
    *   Подсчёт **Среднего FPS** за сессию.
    *   Отслеживание показателя **Мин FPS (1% Low)** — ключевой маркер плавности.
    *   Фиксация **Максимальной** частоты.
    *   Встроенный **счетчик микро-фризов и лагов** для аналитики просадок производительности.
*   **🪐 Симулятор физической нагрузки (Stress Engine)**:
    *   Поток рендеринга физических частиц с поддержкой интерактивной гравитации.
    *   Возможность вызова физического "взрыва" тапом по рендер-боксу.
    *   Ползунок настройки до **12,000+ независимых физических тел**.
    *   Предназначен для искусственного создания троттлинга процессора, чтобы вы могли прямо в приложении увидеть и замерить реакцию счетчика на просадку кадров!
*   **🔴 Эмулирование лагов Main-потока**:
    *   Специальная кнопка искусственного усыпления графического потока на 80 мс для теста точности работы графопостроителя и логера микро-фризов.

## 📲 Как скачать готовый APK? (Инсталляция)

Вы можете получить полностью скомпилированный, готовый к установке файл `.apk` следующими способами:

### Способ 1. Через встроенный экспорт в AI Studio (Самый простой)
1. В интерфейсе сборщика нажмите на выпадающее меню **Settings** (Настройки) или кликните по иконке экспорта в правом нижнем углу.
2. Выберите **Generate APK** (Сгенерировать APK) или **Download ZIP** для скачивания исходного кода на ПК.

### Способ 2. Автоматическая сборка в вашем GitHub (CI/CD)
Мы интегрировали в проект файл конфигурации **GitHub Actions Workflow** (`.github/workflows/android.yml`). 
1. Опубликуйте этот репозиторий на вашем аккаунте GitHub.
2. При каждом вашем `git push` серверы GitHub автоматически запустят компиляцию проекта.
3. Перейдите во вкладку **Actions** в веб-панели GitHub вашего репозитория.
4. Выберите последнюю запущенную сборку и прокрутите вниз до раздела **Artifacts** (Артефакты).
5. Там вы найдете готовый для скачивания файл **`FPS-Monitor-Debug-APK`**!

## 🛠️ Инструкция по ручной установке и запуску

1. Скачайте файл **`app-debug.apk`** на ваше Android-устройство.
2. Откройте файл. Если система выдаст предупреждение о неизвестном источнике, перейдите в настройки системы и включите **"Разрешить установку из этого источника / неизвестных источников"**.
3. Запустите приложение **FPS Monitor**.
4. **Активация Плавающего Счетчик FPS**:
   * Нажмите на главном экране яркую розовую кнопку **"КЛАЦ - АКТИВИРОВАТЬ В НАСТРОЙКАХ"**.
   * Произойдет переход в системный раздел "Специальные Возможности".
   * Зайдите в меню **"Скачанные приложения"** (или "Дополнительные услуги / Услуги").
   * Выберите службу **`FPS Monitor Counter`** и нажмите переключатель **"Включить"**.
5. На экране устройства появится компактный неоновый виджет с текущим FPS. Вы можете запускать любую игру (например, *PUBG, Genshin Impact* или *Roblox*) — виджет останется поверх игры и покажет реальный фреймрейт.

## 💻 Ручная компиляция исходного кода

Если вы хотите собрать проект самостоятельно на компьютере:

### Системные требования:
*   Установленная среда разработки **Android Studio** (Koala / Ladybug или новее).
*   Версия **JDK 17** (рекомендуется Zulu OpenJDK 17).
*   Android-устройство или эмулятор с версией API 26+ (Android 8.0 и выше).

### Команды для терминала:
```bash
# Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/fps-monitor.git
cd fps-monitor

# Дайте права на выполнение Gradle-скрипту (для macOS/Linux)
chmod +x gradlew

# Запустите быструю компиляцию тестовой версии APK
./gradlew assembleDebug
```
После успешного завершения команды готовый файл будет находиться по пути:
`app/build/outputs/apk/debug/app-debug.apk`

## 🛠️ Стековая архитектура проекта (Tech Stack)

*   **Jetpack Compose**: Полностью реактивный и анимированный интерфейс в стиле Киберпанк.
*   **Android Accessibility API**: Системный API для высокопроизводительного наложения окон.
*   **Choreographer Frame Callbacks**: Использование оригинальных системных тиков отрисовки экрана Android (`V-Sync`) для вычисления миллисекундных таймингов без загрузки чипа фоновыми процессами.
*   **Kotlin Coroutines & Flow**: Асинхронное обновление кэша истории графиков производительности.
